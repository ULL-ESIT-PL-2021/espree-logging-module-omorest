## Installation

**Install locally in project**

```console
npm i -D @alu0101209067/addlogging
```

**Install globally**

```console
npm i -g @alu0101209067/addlogging
```

## Usage

### Commands

- **add-logging**

  This command is a executable and show the help about how works.

  ```console
  ~ add-logging
  ```

  ```console

  Páctice 3 npm module addlogging

  Options:
  ```
